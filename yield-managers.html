Yield managers
Strategic control of inventory & rate plans while influencing maximum profit for sold accommodations.

Octabytes's Yield manager is designed and developed with one single purpose – to allow revenue managers to increase and decrease hotel's rates according to demand. Yield manager simplifies changes of rate values in different periods using rate plans – levels within yield group using color coded graphical interface that is easy to monitor and even easier to manage.You can combine yield manager with our channel manager, so you can send your changes to all connected channels at the same time.

Result of revenue management was never easier to implement.

Yield manager features at a glance:

  • Create new levels – rate plans
  • Link levels or keep them independent
  • Change levels by mouse or keyboard
  • Have as many groups as you wish
  • Manage levels by name and colors
  • Visualize your changes
  • Setup yield groups
  • Yield whole year or just one month
  • Yield on room level

Link Manager

Link Manager is a real time saver by allowing to speed up rate plan updates to minimum effort necessary, up to 80% compared to a regular working day without it. By updating just a master rate plan linked rates will be automatically changed according to pre-defined settings.

You can also link between web rate plans and extranets thus making changes a lot easier and keeping again rate parity at all time. With smart Link manager your sales department daily efforts will be dramatically reduced to minimum.

Link manager features at a glance:

  • Interlink rate plans
  • Set mathematical dependencies:
    • 1. In percentage (%)
    • 2. In fixed value
    • 3. Decrease & increase values
  • Insert value once - change many rate plans
  • Change by season
  • Change by day
  • Batch fill values
  • Works on rate import
  • Safety lock
  • Chain rate plans
